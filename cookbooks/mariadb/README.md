# mariadb-cookbook

Install MariaDB 5.5 from SCL repository for CentOS6.5.

## Supported Platforms

CentOS6.5

## Attributes

None

## Usage

### mariadb::default

Include `mariadb` in your node's `run_list`:

```json
{
  "run_list": [
    "recipe[mariadb::default]"
  ]
}
```
